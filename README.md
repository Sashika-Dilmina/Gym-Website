# Gym-Website
A modern and responsive gym promotional website designed to showcase services and attract new members. It features a clean UI, key sections like services and contact, and a fully functional inquiry form for user engagement.


 Live Demo





 Features

    Responsive modern UI (React + Tailwind CSS)

    Sticky navbar with scroll effect (transparent → black)

    Multiple pages (Home, About, Trainers, Services, Pricing)

    Contact form with success message

    Footer with navigation links

 Tech Stack

    React (Vite)

    Tailwind CSS

    React Router DOM

    Lucide React Icons

 Installation & Setup

# Clone repository
git clone https://github.com/Sashika-Dilmina/Gym-Website/tree/main/gym-website

# Go to project folder
cd gym-website

# Install dependencies
npm install

# Start development server
npm run dev
